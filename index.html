About Me

I am an undergraduate computer science major at the University of Kentucky. I am currently employed as a research intern under Dr. Nathan Jacobs, working on the problem aerial image stitching. Previously I worked under Dr. Brent Seales, with the majority of my time being dedicated to the EDUCE project, working to extract data from Herculaneum scrolls.

Projects

Ray Tracer

This was the final project for CS535: Intermediate Computer Graphics. The requirements were to implement a ray tracer in software that was able to display spheres, triangles, and texture triangles using the Lambertian lighting model.

Tabletop Game Simulation

This was a group project for CS499: Senior Design. It involved working with a "customer" to meet project requirements. There were four of us on the project, and we chose to create a game using "modern" OpenGL in order to better familiarize ourselves with the API. We worked to design, prototype, and finally implement what we could of the game.

2D Physics Demo

This was my own project meant to serve as the basis for a 2D Game Engine. It never got that far, but I got the chance to familiarize myself with XNA and implement approximate Newtonian physics. I accomplished this using the Separating Axis Theorem to check collisions between convex objects.

Recommended Books

  • Physically Based Rendering: From Theory to Implementation, Matt Pharr and Greg Humphreys. This book covers the topic of photo-realistic rendering in-depth. The content is presented in a code-centric fashion that easily relates theory to practice.
  • 3D Math Primer for Graphics and Game Development, Fletcher Dunn and Ian Parberry. Dunn and Parberry made great effort to make this book as readable as possible. Thus, there are probably better reference materials available, but the book is wonderful at teaching the basics of the math behind the graphics pipeline.
